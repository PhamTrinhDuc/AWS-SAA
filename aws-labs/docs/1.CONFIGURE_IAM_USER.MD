### PHẦN 1: Dùng tài khoản Root tạo Admin User

1.  **Đăng nhập Root:** Truy cập [aws.amazon.com](https://www.google.com/search?q=https://aws.amazon.com), chọn **Root user** và đăng nhập bằng Email bạn dùng để đăng ký.
2.  **Vào IAM:** Gõ "IAM" trên thanh tìm kiếm và chọn dịch vụ **IAM**.
3.  **Tạo User:**
      * Vào menu **Users** -\> chọn **Create user**.
      * **User name:** Đặt tên là `MyAdmin` (hoặc tên bạn).
      * Tích chọn: **Provide user access to the AWS Management Console**.
      * Chọn: **I want to create an IAM user** (để quản lý mật khẩu thủ công).
      * **Console password:** Chọn *Custom password* và nhập mật khẩu của bạn.
      * Bỏ tích: *Users must create a new password at next sign-in* (để bạn dùng luôn mật khẩu vừa tạo).
4.  **Gán quyền Admin:**
      * Ở trang Permissions, chọn **Attach policies directly**.
      * Trong danh sách hiện ra, tìm và tích vào ô: **AdministratorAccess** (Đây là quyền cao nhất).
5.  **Hoàn tất:** Nhấn **Create user**.
      * **QUAN TRỌNG:** AWS sẽ hiện một cái **Console sign-in URL**. Hãy copy link này lại (nó có dạng `https://[con-so-id].signin.aws.amazon.com/console`). Đây là lối vào riêng cho các User bạn tạo. Cần thận hơn thì tải file .csv chứa các thông tin của Admin vừa tạo

6.  **Tạo MFA cho Admin**
    * Chọn vào User Admin vừa tạo -\> tìm tới `Multi-factor authentication (MFA)` và chọn `Assign MFA device`
    * Đặt tên cho **MFA device name** -\> **MFA device** chọn **Authenticator app** -\> Quét mã QR

### PHẦN 2: Bảo mật tài khoản Root và "Nghỉ hưu" nó 

> Nếu đã tạo MFA trước đó rồi thì bỏ qua bước này mà `signout` luôn

1.  Vẫn ở tài khoản Root, nhấn vào **tên tài khoản** ở góc trên cùng bên phải -\> chọn **Security credentials**.
2.  Tìm phần **Multi-factor authentication (MFA)** -\> nhấn **Assign MFA device**.
3.  Chọn **Authenticator app** (dùng Google Authenticator hoặc Authy trên điện thoại).
4.  Quét mã QR và nhập 2 mã số hiện trên điện thoại để hoàn tất.
5.  **ĐĂNG XUẤT** khỏi tài khoản Root. Từ nay bạn sẽ không dùng nó nữa.


### PHẦN 4: Dùng Admin User để tạo thêm User phụ (nếu cần)

Bây giờ, bạn đóng vai trò là "Quản lý tòa nhà" (Admin):

1. Trên trang Login của AWS chọn IAM User 
2. Đăng nhập với Username: 12 số trong `sign-in URL` có dạng `123958296802` và Password đã tạo
3. Lúc này thay vì dùng Root, bạn sẽ dùng Admin với toàn quyền như Root
4.  Để tạo thêm một User phụ (ví dụ: `Developer-01` chỉ được dùng S3):
      * Lặp lại các bước tạo User như ở Phần 1.
      * Ở phần gán quyền (Permissions), thay vì chọn *AdministratorAccess*, bạn tìm `AmazonS3FullAccess`.
      * Nếu muốn lấy **Access Key/Secret Key** cho User này:
          * Sau khi tạo xong, bấm vào tên User đó -\> tab **Security credentials** -\> kéo xuống **Access keys** -\> nhấn **Create access key**.
          * a. Tích chọn dòng đầu tiên **Command Line Interface (CLI)**. AWS sẽ hiện một cảnh báo nhắc nhở về bảo mật, bạn cứ tích xác nhận và nhấn **Next**.
          * b. **Set description tag (Tùy chọn)**: Bạn có thể đặt tên gợi nhớ như "Key cho máy tính ca nhan" để sau này biết key này dùng ở đâu.
          * c. Retrieve access keys: Đây là bước quan trọng nhất. AWS sẽ hiển thị:
            * *Access Key ID*: Dãy chữ in hoa.
            * *Secret Access Key*: Dãy ký tự bảo mật.
        * d. Download file CSV để lưu lại thông tin

### PHẦN 5. Cấu hình CLI 
1. **Cài đặt AWS CLI**: Nếu chưa cài, bạn cần tải và cài đặt bộ AWS CLI từ trang chủ AWS
2. **Chạy lệnh cấu hình**: Nhập lệnh:
```bash
aws configure
```
3. **Nhập thông tin**: Hệ thống sẽ lần lượt hỏi và bạn dán thông tin từ file .csv vào:
* `AWS Access Key ID [None]`: (Dán Access Key ID của bạn)
* `AWS Secret Access Key [None]`: (Dán Secret Access Key của bạn)
* `Default region name [None]`: (Ví dụ: ap-southeast-1 cho Singapore hoặc us-east-1)
* `Default output format [None]`: (Nhập json)
